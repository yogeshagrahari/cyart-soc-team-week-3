## Learning Objectives
1. Correlate and enrich logs using Elastic Security
2. Integrate threat intelligence feeds with Wazuh + OTX
3. Escalate incidents through TheHive with SITREPs
4. Triage alerts and validate IOCs with VirusTotal
5. Collect and preserve digital evidence using Velociraptor
6. Simulate a full attack-to-report SOC workflow
